# Git-hub-users-data

-Provide a search input box & button to SEARCH for git hub user 
- Up on search Call the below API
      1. avatar_url
      2.followers
      3.following
      4.created_at (Date)
      5.repos_url
Display all the Repos as button , up on click navigate the user to the respective page.
